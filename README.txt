########################### AIDE ###########################

Quelques informations pour comprendre le fonctionnement
du logiciel :

- Ce logiciel permet de récupérer les musiques des niveaux
de Osu! en les copiant et en les renommant correctement dans
un dossier que vous pourrez choisir. Vous pourrez récupérer
des milliers de musiques à écouter en peu de temps !

- Lancer le fichier .bat pour lancer le programme.
Si votre système ne permet pas l'utilisation d'un .bat, 
lancer le fichier .py avec "python OsuTransfert.py" dans 
une console système.

- Les musiques ayant déjà été traitées par le programme ne 
seront plus traitées, à moins que leurs noms soit effacer de
"music.txt".
Si vous voulez que toutes les musiques soient traitées au
début du programme, vous pouvez supprimer le fichier 
"music.txt".
